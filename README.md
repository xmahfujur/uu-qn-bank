# UU QN Bank

A centralized platform for Uttara University students to access previous exam questions, organized by department, batch, semester, and course.

**Live Demo:** https://uuqnbank.vercel.app/

---

## About

UU QN Bank is designed to make previous exam questions easily accessible for Uttara University students. Instead of searching through Facebook groups, Messenger chats, Google Drive folders, or personal collections, students can find all available question papers in one organized platform.

The project aims to simplify exam preparation by providing a structured and user-friendly experience.

---

## Features

* Department-wise navigation
* Batch-wise organization
* Semester-wise question archive
* Course-wise question papers
* Responsive design
* Fast and intuitive interface
* Clean and modern user experience

## Project Structure

    └── Semester

## Project Goals

* Create a centralized archive of previous question papers.
* Help students understand examination patterns.
* Save time by eliminating scattered resources.
* Encourage students to contribute academic materials.
* Build a reliable academic resource for Uttara University.
